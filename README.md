# ScopeSnap - Smart Project Scope Management System

نظام ويب متكامل لتنظيم وتوثيق نطاق العمل في المشاريع التقنية والخدمية، يهدف إلى تقليل سوء الفهم بين صاحب المشروع والمنفذ، ومنع التوسع غير المنظم في المتطلبات.

## 🎯 المشكلة التي يعالجها المشروع

في المشاريع الصغيرة والمتوسطة، يبدأ التنفيذ غالباً قبل أن يكون نطاق العمل واضحاً بالكامل، مما يؤدي إلى:
- اختلاف فهم العميل والمنفذ لما تم الاتفاق عليه
- ظهور طلبات إضافية غير محسوبة
- تأخر التسليم بسبب تغيّر المتطلبات
- صعوبة إثبات ما كان داخل أو خارج الاتفاق
- حدوث خلافات حول السعر أو مدة التنفيذ

## ✨ المميزات الرئيسية

### 1. إدارة المشاريع والعملاء
- إنشاء حسابات المستخدمين (منفذ / عميل)
- إضافة وإدارة بيانات العملاء
- ربط كل مشروع بعميل محدد
- عرض قائمة المشاريع والعملاء

### 2. بناء نطاق العمل المنظم
- إنشاء أقسام متعددة داخل المشروع
- إضافة عناصر تحت كل قسم
- تحديد حالة كل عنصر:
  - ✅ **مشمول (Included)**: سيتم تنفيذه ضمن المشروع
  - ❌ **غير مشمول (Excluded)**: خارج نطاق الاتفاق
  - ⏳ **قيد المراجعة (Pending)**: يحتاج قرار أو توضيح
- كتابة وصف تفصيلي لكل عنصر

### 3. مشاركة النطاق مع العميل
- إنشاء رابط خاص لكل مشروع
- تمكين العميل من فتح الرابط ومراجعة النطاق
- عرض النطاق بطريقة واضحة واحترافية
- إحصائيات شاملة للبنود

### 4. نظام الموافقة الرسمي
- تمكين العميل من الموافقة على النطاق
- تسجيل تاريخ ووقت الموافقة
- حفظ حالة الموافقة داخل المشروع
- منع التعديل العشوائي بعد الموافقة

### 5. إدارة طلبات التغيير
- إنشاء طلبات تغيير جديدة بعد الموافقة
- توضيح نوع التغيير المطلوب
- تحديد حالة الطلب:
  - ⏳ قيد الدراسة
  - ✅ مقبول
  - ❌ مرفوض
- تسجيل سبب القبول أو الرفض

### 6. سجل النشاطات (Timeline)
- تسجيل جميع الأحداث المهمة
- عرض الأحداث مرتبة زمنياً
- تتبع التغييرات والقرارات
- شفافية كاملة في سير العمل

### 7. لوحة تحكم تفاعلية
- عرض إحصائيات شاملة
- متابعة حالة المشاريع
- عرض آخر التعديلات
- ملخص سريع للبنود

## 🛠️ التقنيات المستخدمة

- **React 19** - مكتبة JavaScript لبناء واجهات المستخدم
- **TypeScript** - لغة البرمجة الأساسية
- **Vite** - أداة البناء السريعة
- **Tailwind CSS 4** - إطار عمل CSS للتصميم
- **React Router** - لإدارة التنقل بين الصفحات
- **Lucide React** - مكتبة الأيقونات
- **LocalStorage** - لتخزين البيانات محلياً

## 📦 التثبيت والتشغيل

- Node.js (الإصدار 18 أو أحدث)

1. **تثبيت التبعيات:**

2. **تشغيل المشروع في وضع التطوير:**
```bash

```

3. **بناء المشروع للإنتاج:**
```bash
npm run build
```

4. **معاينة البناء:**

## 🎨 الميزات التصميمية

- **تصميم RTL كامل**: دعم كامل للغة العربية
- **واجهة عصرية**: تصميم حديث وأنيق
- **تجاوب كامل**: يعمل على جميع الأجهزة
- **ألوان دلالية**: ألوان واضحة للحالات المختلفة
- **رسوم متحركة سلسة**: انتقالات سلسة بين الصفحات
- **إمكانية الوصول**: تصميم يراعي سهولة الاستخدام

## 📊 حالات المشاريع

1. **مسودة (Draft)**: المشروع قيد الإعداد
2. **أُرسل للعميل (Sent)**: تم إرسال النطاق للمراجعة
3. **تمت الموافقة (Approved)**: وافق العميل رسمياً
4. **قيد التنفيذ (In Progress)**: جاري العمل على المشروع
5. **مكتمل (Completed)**: تم الانتهاء من المشروع

## 🔄 سير العمل المثالي

1. **إنشاء المشروع**: إضافة مشروع جديد وربطه بعميل
2. **بناء النطاق**: إنشاء أقسام وعناصر واضحة
3. **تحديد الحالات**: تصنيف كل بند (مشمول/غير مشمول/قيد المراجعة)
4. **إرسال للعميل**: مشاركة الرابط مع العميل
5. **مراجعة العميل**: العميل يراجع النطاق
6. **الموافقة الرسمية**: العميل يوافق على النطاق
7. **بدء التنفيذ**: البدء بالعمل وفق النطاق المعتمد
8. **إدارة التغييرات**: تسجيل أي طلبات تغيير لاحقة
